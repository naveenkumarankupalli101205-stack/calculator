# 🧮 Calculator

A simple and responsive Calculator built using **HTML**, **CSS**, and **JavaScript**.

## 🚀 Features

- Addition
- Subtraction
- Multiplication
- Division
- Clear (AC) Button
- Delete (DE) Button
- Decimal Support
- Responsive Design
- Modern User Interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
calculator/
│── index.html
│── style.css
└── script.js
```

## ▶️ How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open **index.html** in your web browser.
